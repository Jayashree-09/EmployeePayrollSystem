# Employee Payroll System

## 📌 Project Overview

The Employee Payroll System is a Java desktop application developed using Java Swing and MySQL. It helps organizations manage employee information, calculate salaries, and maintain payroll records through an easy-to-use graphical user interface.

---

## 🚀 Technologies Used

- Java
- Java Swing (GUI)
- JDBC (Java Database Connectivity)
- MySQL
- VS Code
- Git & GitHub

---

## ✨ Features

- Employee Login
- Add New Employee
- View Employee Details
- Update Employee Information
- Delete Employee
- Salary Management
- Payroll Calculation
- Database Connectivity using MySQL
- User-Friendly Graphical Interface

## 🛠 Prerequisites

Before running the project, install the following:

- Java JDK 17 or later
- VS Code (or any Java IDE)
- MySQL Server
- MySQL Connector/J (JDBC Driver)

---

## ⚙️ Database Setup

1. Install MySQL.
2. Create a database:

```sql
CREATE DATABASE payroll_db;
```

3. Import the SQL file (if available).

4. Update the database connection details in the project:

```java
url = "jdbc:mysql://localhost:3306/payroll_db";
username = "root";
password = "your_password";
```

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Jayashree-09/EmployeePayrollSystem.git
```

2. Open the project in VS Code.

3. Add the MySQL JDBC Driver to the `lib` folder.

4. Configure the database connection.

5. Compile the project.

6. Run `Main.java`.

The Employee Payroll System application will start.
